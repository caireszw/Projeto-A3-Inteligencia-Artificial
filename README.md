# Projeto-A3-Inteligencia-Artificial

**Objetivo:** Este projeto é um sistema de suporte à decisão clínica que acelera a triagem de doenças cardíacas usando comitê de algoritmos (KNN, SVM, Árvore de Decisão). Ele resolve o atraso no diagnóstico identificando pacientes de risco para apoiar médicos em clínicas e hospitais.
